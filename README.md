# modernews.github.io

### This page is basic guide to typesetting written in Polish

This page's contents are written using [RST](https://docutils.sourceforge.io/rst.html), all commits are automatically build github-side and posted [live](https://modernews.github.io). There are no build files stored locally.  
If you want to make/propose changes, translations, etc. repository is public, feel free to create new pull requests or issues.

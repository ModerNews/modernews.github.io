Język angielski

Systemy UNIX - Linux

Od 2022 roku korzystam z różnych dystrybucji systemu Linux, zarówno na urządzeniach personalnych jak i w kontenerach/serwerach obsługujących moje projekty. Potrafię efektywnie korzystać z terminala do zarządzania systemem i projektami. Posiadam również doświadczenie z edytorem Vim. Do dystrybucji, z którymi miałem największe doświadczenie należą Arch Linux, Ubuntu, Debian oraz Alpine Linux. Przez krótki czas korzystałem również z systemu FreeBSD.

Tworzenie aplikacji backendowych - Java, Python

Sprawnie posługuję się językiem Java i frameworkiem Spring (wraz z Spring Boot, Spring Security, etc.) w celu tworzenia aplikacji backendowych. Od roku 2022 tworzę również aplikacje webowe w języku Python z wykorzystaniem frameworków FastAPI i Flask.

Tworzenie aplikacji frontendowych - HTML, CSS i React

Sprawnie posługuję się językami HTML i CSS, dobrze znam framework React, a także powiązane z nim języki JavaScript i TypeScript. Zajmowałem się tworzeniem aplikacji frontendowych małej skali, zarówno w czystych HTML i CSS, jak i w Reactcie, również z wykorzystaniem bibliotek takich jak MUI.

Obsługa baz danych

Dobrze znam język SQL, w swoich projektach miałem okazję posługiwać się dialektami MySQL (MariaDB), PostgreSQL i SQLite. Wliczając w to zarówno przepytywanie baz, jak i ich tworzenie.

Git, Docker, Nginx

Na codzień w swoich projektach posługuję się narzędziem Git, zarówno konsolą jak i GUI dostępnymi w IDE. Dobrze znam Dockera, posługuję się zarówno Dockerfile'ami jak i docker-compose. Znam podstawy konfiguracji serwera Nginx.

Programowanie

Biegle posługuję się nie tylko językami Java i Python, ale również JavaScript, TypeScript, Rust, SQL bash i awk. Znam także podstway języka lua.
